# Salesforce-LWC-Learning
This repository maintains all the LWC learning concepts 
We have component with following concepts implemented
1. Lifecycle Hooks
2. Decorators
3. Component Communication
   1. Parent to child (using @api decorator)
   2. Child to parent (using custom Events)
   3. Inter component Commincation (using LMS)
4. Wire Vs Imperative Apex class (Invoice Pagination which is having 10000 invoice records)
